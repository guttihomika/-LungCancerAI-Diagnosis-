# 🫁 LungCancerAI - Early Lung Cancer Detection

## 🚀 Features
- Upload chest X-ray → AI diagnosis in 3 seconds
- 87%+ accuracy (simulated CNN model)
- Malignant/Benign classification
- Confidence scoring
- Drag & drop interface

## 🛠️ Tech Stack
**Frontend:** React 18 + Tailwind + Framer Motion  
**AI Backend:** TensorFlow.js (simulated)  
**Deployment:** Vercel/Netlify
